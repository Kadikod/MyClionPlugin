# MyClionPlugin
**Автор:** Мороз Катерина Джулия  **Группа:** М3123

## Описание плагина
Плагин для CLion, который добавляет тестовую функциональность через меню Tools.

## Возможности
- Добавление пункта меню "Say Hello" в раздел Tools
- Отображение приветственного сообщения при активации

## Установка
1. Склонировать репозиторий.
2. Собрать проект через Gradle.
3. Установить полученный JAR-файл через Settings → Plugins.
